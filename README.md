# Gravity Wells & Missiles

A 2D space artillery game where two players battle through an asteroid field using gravity physics and homing missiles.

## Play Now
https://JoeHowarth.github.io/gravity-wells-n-missiles/

### Run locally
```bash
npm run dev
```
